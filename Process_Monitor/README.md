# Process Monitor (Python)

A lightweight system monitoring tool built with Python that allows users to view running processes, monitor CPU and memory usage, visualize system resource usage in real-time graphs, and terminate processes directly from the interface.

## Features

- Real-time process monitoring
- CPU and Memory usage tracking
- Interactive system resource graphs
- Process-specific resource monitoring
- Fast process search functionality
- Ability to terminate running processes
- Lightweight Tkinter GUI

## Technologies Used

- Python
- Tkinter (GUI)
- psutil (System monitoring)
- Matplotlib (Graph visualization)

## Project Architecture

GUI Layer
- Built using Tkinter for user interaction

Monitoring Layer
- Uses psutil to collect system and process statistics

Visualization Layer
- Matplotlib animations used for real-time CPU and memory graphs
