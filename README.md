A lightweight **system monitoring desktop application** built with Python that allows users to view running processes, track CPU and memory usage, visualize system performance in real-time graphs, and manage processes directly from an interactive GUI.

---

## 🚀 Features

- 📊 **Real-Time System Monitoring**
  - Track CPU and memory usage dynamically.

- 🧾 **Process Listing**
  - View all active system processes with:
    - Process ID (PID)
    - Process Name
    - CPU Usage
    - Memory Usage

- 🔎 **Fast Process Search**
  - Quickly filter processes using a search bar.

- 📈 **System Resource Graphs**
  - Real-time visualization of system CPU and memory usage.

- 📉 **Process-Level Monitoring**
  - Monitor CPU and memory consumption of a selected process.

- ⚡ **Process Management**
  - Terminate selected processes directly from the GUI.

- 🖥️ **User-Friendly Interface**
  - Built with a simple and clean Tkinter-based desktop interface.
